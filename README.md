# Test Assignment

## Instructions

### 1. Clone repository

```bash
git clone https://github.com/OlgaReznik/payment-qa-test.git
```

### 2. Install Dependencies
This project uses Node.js and Playwright. You need to install the required dependencies using npm:

```bash
npm install
npx playwright install
npm install axios
```

### 3. Running tests

```bash
npx playwright test
```
